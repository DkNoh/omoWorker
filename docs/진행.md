## 엔진 개선

- [X]  **baseQuery DRY** — MapperXmlTemplate가 `<sql id="baseQuery">` 1회 정의 후 `<include>` 참조. rawQuery 3회 인라인→1회
- [X]  **쿼리 시그니처 자동 주입** — MapperXmlTemplate 각 select에 `/* {Domain}Mapper.{method} */` 생성
- [X]  **JsTemplate badge formatter 자동화** — ColumnConfig.optionsText(`value:label,...`)가 있으면 `TuiCommon.badgeByValue({ labels })` 자동 생성. tones는 tui-common의 `badgeColorFor`가 라벨 기반으로 자동 부여
- [X]  **JsTemplate 커스텀 dateFormat** — dateFormat이 키워드(AUTO/DATE/DATETIME/NONE)가 아니면 원본 문자열을 `TuiCommon.formatDate(value, '<포맷>')`로 전달. dayjs 대소문자 구분(MM=월, mm=분) 보존을 위해 키워드만 대문자 정규화, 그 외는 원본 유지
- [ ]  **컬럼 헤더 한글화** — ColumnConfig.headerLabel 옵션. 현재 영문 컬럼명 그대로 출력. 사용자가 "나중에 한 번에 한글화" 예정
